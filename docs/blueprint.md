# **App Name**: FoamPilot

## Core Features:

- Case Upload: Allow users to upload OpenFOAM case directories via drag and drop or file selection.
- Case Selection: Enable selection of uploaded OpenFOAM case directories for active management.
- Dictionary Editor: Provide an interactive editor for modifying common OpenFOAM dictionary files.
- Simulation Launcher: Configure and launch OpenFOAM solvers on remote servers.
- Console Monitoring: Display real-time output from running OpenFOAM solvers in a console.
- AI Parameter Optimization: AI-powered tool suggests optimal solver parameters based on the case setup. The AI tool analyzes the case and recommends parameter settings for the chosen solver. The tool also checks for required files.
- Case Creation: Allow users to create new directories and OpenFOAM case files.
- Tutorial Cases: Provide an option to run cases directly from OpenFOAM tutorials.

## Style Guidelines:

- Primary color: A deep, but vibrant blue (#3F51B5), reminiscent of engineering blueprints, and indicating reliability.
- Background color: A very light grey (#F0F2F5) to ensure readability and minimize eye strain during long sessions.
- Accent color: A bright yellow-green (#AEEA00) to highlight important actions and data points.
- Additional color: Red (#FF0000) to indicate errors, warnings, or critical alerts.
- Font pairing: 'Inter' (sans-serif) for both headlines and body text, for a clean, modern look and excellent readability.
- Lightweight and modern design with optional acrylic or translucent parts.
